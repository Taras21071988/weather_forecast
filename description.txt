Импортируем модуль conditions из файла conditions.js.

Получаем ссылки на элементы на странице - форма, поле ввода и заголовок.

Определяем функцию renderCard, которая отображает карточку на странице, принимая HTML-разметку в качестве параметра.

Определяем функцию removeCard, которая удаляет предыдущую карточку с погодой, если она существует.

Определяем функцию showError, которая создает карточку с сообщением об ошибке ввода города.

Определяем функцию showCard, которая создает карточку с информацией о погоде на основе переданных данных.

Определяем функцию getWeather, которая асинхронно получает данные о погоде с помощью API, используя полученный город.

Устанавливаем обработчик события отправки формы. При отправке формы вызывается асинхронная функция, которая получает данные о погоде для указанного города. Если полученные данные содержат ошибку, вызывается функция showError, в противном случае создается карточка с информацией о погоде и вызывается функция showCard для ее отображения.
